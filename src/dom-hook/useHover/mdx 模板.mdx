# useHover

检测鼠标是否悬停在元素上。钩子返回一个 ref 和一个布尔值，指示当前是否悬停具有该 ref 的元素。只需将返回的 ref 添加到要监视其悬停状态的任何元素即可。

useHover 此方法的一个潜在错误：如果您有更改添加到的元素的逻辑，则事件侦听器不一定会应用于新元素。使用 useCallbackHover 替代
